# KanBan_Board  👨‍💻 

## Used technology stack

* TypeScript
* React
* HTML 5
* Bootstrap 5
* Redux-Toolkit
* ESLint
* Prettier
* React-Bootstrap
* React-Testing-Lib

## Setup
To run this project

```
$ npm install

```
Create .env file following the .env-example

```
$ npm run start

```
To run the tests
```
$ npm run test

```
